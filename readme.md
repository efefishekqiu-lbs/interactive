# Interactive – Publisher Website  

En frontend-webbplats för företaget **Interactive** som publicerar 2 olycka spel.
Byggd med HTML, CSS, JavaScript och jQuery – utan backend.

---

## Arbetsprocess  

- Byggde sidan med HTML och CSS.  
- Lade till interaktivitet med JavaScript och jQuery.  
- Testade responsivitet och förbättrade koden.  

---

## Tekniska val  

- **HTML5** – Struktur  
- **CSS3** – Design & responsiv layout  
- **JavaScript** – Funktionalitet  
- **jQuery** – DOM ändringar och effekter  

Valde ren frontend eftersom webbplatsen är statisk.

---

## Självutvärdering  

**Styrkor:**  
- Tydlig struktur  
- Responsiv design    

**Förbättringar:**  
- Bättre tillgänglighet  
- Möjlig backend i framtiden
